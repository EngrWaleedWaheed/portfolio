Project Overview

This project demonstrates the design and implementation of a scalable enterprise campus network using Cisco Packet Tracer. The network is divided into multiple departments using VLANs while dynamic routing, security, and internet access are configured through enterprise networking protocols.

Network Topology

Enterprise network designed and simulated in Cisco Packet Tracer.

Key Features

🌐 VLANs

Separate departments into secure broadcast domains.

📡 OSPF

Dynamic routing between multiple routers.

🛡 ACL

Traffic filtering and network security.

🌍 NAT

Internet connectivity using Network Address Translation.

📥 DHCP

Automatic IP assignment for client devices.

🔀 Inter-VLAN Routing

Communication between different VLANs.

Configuration Highlights

Router Configuration

Configured OSPF routing, DHCP pools, static routes and NAT.

Switch Configuration

Created VLANs, trunk links and inter-VLAN routing.

Security

Applied Access Control Lists for traffic filtering.

Troubleshooting

Verified connectivity using ping, traceroute and routing tables.

Challenges & Solutions

Challenge

Configuring communication between multiple VLANs while maintaining network security and proper routing.

Solution

Implemented Inter-VLAN Routing, configured OSPF for dynamic routing, and applied ACLs to control traffic between departments.

Project Results

✔ Successful VLAN Communication

Devices in different VLANs successfully communicated using Inter-VLAN Routing.

✔ Dynamic Routing

OSPF automatically exchanged routing information between routers.

✔ Secure Network

ACLs restricted unauthorized traffic while maintaining connectivity.
